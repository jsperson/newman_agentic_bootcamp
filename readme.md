# Agentic Software Engineering Bootcamp

**Newman University** — Spring 2026

A 6-week bootcamp for professionals and students learning to build production-quality software with AI agents. Not vibe coding — structured, spec-driven development using tools like Claude Code, Cursor, and OpenAI Codex.

**Course page:** [newmanu.edu/ai-software-eng](https://newmanu.edu/ai-software-eng)

## Schedule

Tuesdays, March 24 – April 28, 2026 | 4:30 – 6:00 PM CT | Hybrid (on-campus + MS Teams)

| Week | Topic |
|------|-------|
| 1 | **Foundation & Setup** — Tool installation, vibe coding vs. AI-assisted development |
| 2 | **Dynamic Prompting** — Model selection and workflow optimization |
| 3 | **Plan-First Development** — Specification-to-implementation philosophy |
| 4 | **Development Loop & Debugging** — Full development cycle and testing strategies |
| 5 | **Documentation & Rules** — LLM-optimized specs and agent rules |
| 6 | **Team Dynamics + Capstone** — Code review workflows and legacy modernization project |

## Prerequisites

- Experience with at least one programming language (loops, functions, methods, OOP)
- Basic IDE familiarity (Cursor, VS Code, etc.)
- Command line experience helpful
- Ability to install software on a personal laptop
